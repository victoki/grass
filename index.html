About Me

My name is Victoria. I am a Canadian living in the USA. I worked as a game artist for a decade and have since decided to change careers. I spend my free time outside work and school playing video games and taking care of a few pet reptiles. I'm currently studying java in my free time, and think its a neat language with some weird quirks. For example, all the data types for java are lowercase. Except this guy: String. If you try to declare 'string' it freaks the compiler out, because 'String' is special and absolutely needs to be upper case.

Things I would like to learn:

  • Cleaner Code
  • Making CSS do crazier wizard-things with your webpages
  • More ways to use html5/css/javascript
  • More applications that use all three of the above.
  • How to get my job to give me more time so I don't recycle my old CSS.
